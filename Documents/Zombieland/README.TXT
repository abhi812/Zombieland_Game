PROJECT TITLE: Zombie World
PURPOSE OF PROJECT: Create a simple "twin stick shooter" style zombie survival game
VERSION or DATE: May 9th 2018
HOW TO START THIS PROJECT: Just click run
AUTHORS: Abhishek Kumar Gupta and Chris Harding
USER INSTRUCTIONS: Move the player with W (up),A (left), S( down), D (right) or the Arrow Keys,
and fire by clicking the mouse.
Avoid the zombies for as long as you can!

Each zombie killed will give 50 points.
When zombie in the world is reduced to sufficiently low value, a wave of zombie will come.
Everytime you survive one wave of zombie you'll get 500 bonus point and will level up.
Game finishes when zombie eats the survivor.

Player and Zombie models created by rileygombart of Opengameart.org
Player downloaded from: https://opengameart.org/content/animated-top-down-survivor-player
Zombie downloaded from: https://opengameart.org/content/animated-top-down-zombie
